# Cover

## Battery Cover

### Removal Instructions
1. Power off device completely
2. Remove SIM tray using ejector tool
3. Remove 4x Phillips #00 screws from corners
4. Use plastic opening tool to separate cover
5. Lift battery cover carefully

### Components Under Cover
- Battery
- SIM tray
- SD card slot (if applicable)
- Antenna connections

### Notes
- Work on clean, static-free surface
- Use ESD mat for protection
- Be careful of hidden clips
