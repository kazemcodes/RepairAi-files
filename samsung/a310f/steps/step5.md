# Step 5: Disconnect Battery

## Objective

Disconnect the battery from the main board to prevent short circuits.

## Tools Required

- Plastic spudger
- Tweezers

## Battery Connector Location

Refer to [board.md](../board.md) for connector positions.

- Connector: J1 (Battery connector)
- Location: Bottom left of main board

## Instructions

1. **Locate battery connector** - J1 on main board
2. **Lift connector tab** - Use plastic spudger
3. **Pull gently** - Disconnect flex cable
4. **Verify** - Connector is fully detached

## Warnings

- ALWAYS disconnect battery first
- Never work on powered device
- Avoid metal tools near connectors
- Be gentle with delicate cables

## After Disconnection

- [ ] Battery connector is disconnected
- [ ] Cable is clear of board
- [ ] No exposed contacts touching

## Notes

- Some devices have protective tape
- Remove tape carefully if present
- Don't pull on cable, pull on connector
- Check for glue, use heat if needed
