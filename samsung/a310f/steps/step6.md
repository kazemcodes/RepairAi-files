# Step 6: Remove Shield

## Objective

Remove the main board metal shield to access components.

## Tools Required

- Phillips #00 screwdriver
- Plastic spudger
- Tweezers

## Shield Location

The metal shield covers the main board components:
- CPU/SoC
- RAM
- Flash storage
- Power management IC

## Instructions

1. **Verify screws removed** - From Step 4
2. **Locate shield** - Center of main board
3. **Lift shield** - Use plastic spudger
4. **Work around edges** - Release all clips
5. **Remove shield** - Keep flat to avoid bending

## Components Under Shield

| Component | Function |
|----------|----------|
| CPU/SoC | Main processor |
| RAM | Memory |
| Flash | Storage |
| PMIC | Power management |

## Notes

- Handle shield carefully
- Don't bend or deform
- Keep track of thermal pads
- Note pad positions for reassembly
