# Step 3: Remove Back Cover

## Objective

Remove the battery cover to access internal components.

## Tools Required

- Phillips #00 screwdriver
- Plastic opening tool
- Suction cup
- SIM ejector tool

## Instructions

### 1. Eject SIM Tray

1. Locate SIM tray on side of device
2. Insert ejector tool into small hole
3. Push gently until tray ejects
4. Remove SIM tray and set aside

### 2. Remove Screws

1. Locate 4x Phillips #00 screws on back cover
2. Remove all 4 screws
3. Place screws on magnetic mat in order

### 3. Separate Cover

1. Insert plastic opening tool between cover and frame
2. Work around edges to release clips
3. Use suction cup to help lift
4. Carefully separate cover from device
5. Remove cover completely

## Components Exposed

- Battery
- Main board
- Antenna flex cables
- Speaker module

## Notes

- Work slowly to avoid breaking clips
- Listen for clicking sounds
- Check for hidden screws
- Don't force the cover off
