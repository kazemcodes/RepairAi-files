# Step 9: Install Display

## Objective

Install the new display assembly and connect all cables.

## Tools Required

- New display assembly
- Adhesive (B7000 or equivalent)
- Plastic spudger
- Tweezers
- Cleaning supplies

## Pre-Installation

1. **Clean surfaces** - Remove old adhesive
2. **Check new display** - Inspect for damage
3. **Test connectors** - Ensure they work
4. **Prepare adhesive** - Apply to frame

## Instructions

### 1. Apply Adhesive

1. Apply thin layer of adhesive to frame
2. Cover edges evenly
3. Wait 30 seconds for tackiness

### 2. Position Display

1. Align display carefully
2. Start from top edge
3. Lower into position
4. Check alignment

### 3. Connect Cables

**Touch Panel (J3)**
1. Insert flex cable into connector
2. Push down locking tab
3. Verify secure connection

**Display (J2)**
1. Insert flex cable fully
2. Push down locking tab
3. Verify secure connection

### 4. Secure Display

1. Press gently around edges
2. Allow adhesive to set
3. Check for gaps

## Notes

- Don't over-tighten connections
- Keep connectors clean
- Align perfectly before pressing
- Check display works before sealing
