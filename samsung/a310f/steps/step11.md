# Step 11: Test

## Objective

Test device functionality before final assembly.

## Instructions

### 1. Power On

1. Connect battery (if not connected)
2. Press power button
3. Watch for boot logo
4. Wait for home screen

### 2. Display Test

- [ ] Screen displays correctly
- [ ] No dead pixels
- [ ] Colors look normal
- [ ] Brightness works
- [ ] No lines or artifacts

### 3. Touch Test

- [ ] Touch responsive
- [ ] Multi-touch works
- [ ] Edge response
- [ ] No ghost touches

### 4. Hardware Test

| Component | Test |
|-----------|------|
| Power button | Press to wake/sleep |
| Volume buttons | Adjust volume |
| Home button | Press to return home |
| Speaker | Play sound |
| Microphone | Record audio |
| Camera | Take photo |
| WiFi | Connect to network |
| Bluetooth | Pair with device |
| Sensors | Check proximity, light |

### 5. Charging Test

- [ ] Charges via USB
- [ ] Shows charging indicator
- [ ] Battery percentage increases

## If Problems Occur

- Display issues: Check connectors
- Touch issues: Clean connectors
- No power: Check battery connection
- Other: Disassemble and check

## Notes

- Don't apply pressure to screen
- Run multiple tests
- Check all functions
- Document any issues
